# Vineeta19-prog 📝

A client-side **Task Manager Application** built with **HTML**, **CSS**, and **JavaScript**.  
This project allows users to add, delete, and toggle tasks with persistent storage using the **localStorage API**.  
It features a clean, responsive UI with visual indicators for completed and incomplete tasks.

---

## 📂 Project Structure

- **index.html**  
  The main HTML file that defines the structure of the task manager.

- **style.css**  
  Contains all the styling rules, including gradients, rounded components, and responsive design.

- **script.js**  
  Implements the core functionality:
  - Add new tasks
  - Delete tasks
  - Toggle task completion state
  - Save tasks to localStorage for persistence

- **checked.jpg / unchecked.png**  
  Icons used to visually represent task states (completed vs incomplete).

- **circle.jpg / to-do image.jpg**  
  Additional UI assets for styling and enhancing the user interface.

---

## 🚀 Features

- ✅ Add new tasks easily  
- ❌ Delete tasks when no longer needed  
- 🔄 Toggle between completed and incomplete states  
- 💾 Persistent storage using **localStorage**  
- 🎨 Clean, responsive UI with gradients and rounded components  

---

## 🛠️ Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling and layout  
- **JavaScript (ES6)** – Logic and interactivity  
- **localStorage API** – Persistent data storage
